hookdepth {hookdepth}R Documentation

plot hook depth

Description

compute longline hook profiles and make a plot

Usage

hookdepth(main,branch,number,inter,vspeed,hspeed,log=TRUE,...)

Arguments

main

length of floatline

branch

length of branchline

number

hook number per basket

inter

distance of hooks

vspeed

vessel speed

hspeed

shooting speed

log

ditance (T) or time (F)

...

arguments to be passed to plot.default()

Examples

hookdepth(25,26,26,33,9.6/2,6)
